# AutoAnswer (自动答题助手) 

AutoAnswer 是一款强大的油猴（Tampermonkey）脚本，能够自动识别网页中的测试题目，调用 AI（如 DeepSeek、OpenAI 等）生成答案，并将答案直接内联展示或自动填入页面中。支持选择题、判断题、填空题和问答题的自动作答。



---

## 🌟 核心功能

* **智能题目识别**: 采用多重策略（平台专用选择器、表单组检测、结构化列表、文本启发式规则）精准提取网页中的题目和选项。
* **全题型支持**: 完美兼容 **单选题、多选题、判断题、填空题（包含多空）** 以及 **问答题（简答、论述）**。
* **自动填写与高亮**:
* 选择题/判断题：自动点击选项并高亮正确答案。
* 填空题/问答题：自动将答案填入 Input、Textarea 甚至富文本编辑器中。


* **多模型与备用 API**: 默认推荐接入 DeepSeek，兼容 OpenAI 格式。支持配置备用 API，主线路失败时自动无缝切换。
* **图片识别 (Vision) 支持**: 开启后可自动抓取题目中的图片并发送给多模态大模型（需配合 GPT-4o 等支持视觉的模型使用）。
* **现代化悬浮面板**: 提供可拖拽的操作面板，支持实时进度展示、运行日志查看以及可视化配置。
* **SPA 页面兼容**: 自动监听路由变化，单页面应用切换无缝重新扫描。

## 📦 安装指南

1. **安装脚本管理器**: 在浏览器中安装 [Tampermonkey](https://www.tampermonkey.net/)（油猴）扩展程序。
2. **安装脚本**: 新建一个脚本，将 `AutoAnswer.js` 的源码复制进去并保存，或者通过脚本发布链接直接安装。
3. **准备 API Key**: 您需要拥有 DeepSeek 或 OpenAI 等大模型厂商的 API Key。
* [DeepSeek 开放平台](https://platform.deepseek.com/) (推荐，高性价比)



---

## 🚀 使用方法

脚本加载成功后，页面右下角（默认）会出现一个半透明的悬浮控制面板。

1. **配置 API**: 点击面板上的 **⚙ (设置)** 图标，填入您的 API 提供商、模型名称以及 `API Key`。
2. **扫描题目**: 点击 **“扫描”** 按钮。脚本会高亮页面中识别到的所有题目，并在面板上显示题数。
3. **开始答题**: 点击 **“答题”** 按钮。脚本将按照设定的延迟时间，逐题请求 AI 并自动将答案注入页面。
4. **查看日志**: 如果遇到问题，可点击 **📋 (日志)** 图标查看详细的运行日志和报错信息。
5. **清除结果**: 点击 **“清除”** 按钮可移除所有 AI 注入的徽章、高亮和填入的文本。

## ⚙️ 设置说明

在设置面板中，您可以进行深度定制：

| 设置项 | 说明 | 默认值 |
| --- | --- | --- |
| **API 提供商** | 支持 DeepSeek、OpenAI 或自定义（需兼容 OpenAI 接口格式）。 | DeepSeek |
| **API 端点** | API 的请求地址。 | `https://api.deepseek.com/v1/chat/completions` |
| **模型名称** | 调用的具体大模型名称。 | `deepseek-chat` |
| **API Key** | 您的身份验证密钥。 | - |
| **备用 API** | 当主 API 失败（如超时、限流）时自动使用的备用节点与密钥。 | - |
| **每题延迟 (ms)** | 两次请求之间的间隔时间，防止触发 API 限流。 | `800` |
| **最大题数** | 单次扫描支持的最大题目数量。 | `50` |
| **重试次数** | 单个题目请求失败后的最大重试次数。 | `1` |
| **模糊匹配阈值** | AI 返回纯文本时，与选项进行文本相似度匹配的百分比阈值。 | `50%` |
| **启用图片识别** | 抓取题目内的图片并发送给 AI（需配置支持多模态的模型）。 | `关闭` |

## ⌨️ 快捷键

为了提升效率，脚本内置了全局快捷键支持：

* `Alt + S` : 快速扫描题目
* `Alt + A` : 自动开始答题
* `Alt + C` : 清除所有 AI 答案和高亮
